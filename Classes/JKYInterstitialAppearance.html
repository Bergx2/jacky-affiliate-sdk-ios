JKYInterstitialAppearance Class Reference

Inherits from NSObject
Declared in JKYInterstitialAppearance.h

Overview

This is a simple value class that represents the visual appearance of an interstitial.

You can either set a custom appearance globally (see [JKYAffiliateManager setDefaultInterstitialAppearance:]), or override the default by specifying an instance of this class when showing an interstitial (see [JKYAffiliateManager showInterstitialWithAppearance:completionBlock:]).
