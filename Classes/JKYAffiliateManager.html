Overview

This class is the main entry point into the SDK.

Setup

To use the SDK to show any interstitials, you first have to identify your app.

In your app delegate’s application:didFinishLaunchingWithOptions:, call startWithAPIKey:andAPISecret: to setup the Jacky Affiliate SDK:

[[JKYAffiliateManager sharedManager] startWithAPIKey:@"<YOUR API KEY>"
                                        andAPISecret:@"<YOUR API SECRET>"];

That’s all you have to do. You can find your API key and secret in the Jacky Affiliate dashboard.

Showing interstitials

Once the SDK is setup, you can start showing interstitials in your app.

In the most simple case, you can just use showInterstitial to show an interstitial with the default appearance in the view of your application’s key window’s root view controller (which is what you usually want to do). If, for some reason, you want to have specify a different superview for the interstitial, you can do that using the showInterstitialInView: method.

Without further configurations, interstitials are displayed with the default appearance, which looks like this:

If you want, you can customize certain aspects of that appearance, like the border color and width. Interstitial appearances are represented by instances of JKYInterstitialAppearance and can be configured globally, as well as overridden for individual interstitials.

To configure a custom interstitial appearance globally, see defaultInterstitialAppearance. Alternatively (or additionally), you can use showInterstitialWithAppearance: (or showInterstitialInView:withAppearance: resp.) to override the global default for that individual interstitial instance.

Properties

defaultInterstitialAppearance

Set a custom default interstitial appearance.

@property (nonatomic, strong) JKYInterstitialAppearance *defaultInterstitialAppearance

Discussion

You can specify a custom default instance of JKYInterstitialAppearance, that will be used as the default when showing interstitials. Of course, you can still override this default for individual interstitials via showInterstitialWithAppearance: (or showInterstitialInView:withAppearance: resp.).

The predefined default interstitial appearance defines a 5px black border.

testModeEnabled

Enable or disable test mode. This mode is intended for testing purposes to get you started quickly with Jacky.

@property (nonatomic) BOOL testModeEnabled

Discussion

If test mode is enabled, only demo interstitials will presented. No partnership or further configuration of your affiliate account are necessary.

Defaults to NO.

Please note: Even with test mode enabled, you still need a Jacky affiliate account and call startWithAPIKey:andAPISecret: with your API key and API secret!
